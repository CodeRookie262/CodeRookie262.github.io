# 定义Promise类

Promise 对象是由关键字 new 及其构造函数来创建的。该构造函数会把一个叫做“处理器函数”（executor function）的函数作为它的参数。这个“处理器函数”接受两个函数resolve 和 reject作为其参数。当异步任务顺利完成且返回结果值时，会调用resolve 函数；而当异步任务失败且返回失败原因（通常是一个错误对象）时，会调用reject 函数。

然后我们再来看下Promise/A+规范:

# Promise有3种状态:
  1. pending(进行中): 初始状态，既不是成功，也不是失败状态
  2. fulfilled(成功): 意味着操作成功完成
  3. rejected(失败): 意味着操作失败。

那么就先封装下基础版Promise,封装方式采用闭包封装

var Promise = (function(){
    //定义状态
    var PENDING = "pending",
        FULFILLED = "fulfilled",
        REJECTED = "rejected";
    function MyPromise(executor){
        var that = this;
        this.value = PENDING;//当前状态,默认进行中
        this.value = undefined;//成功的值
        this.reason = undefined;//失败的值
        this.queue = [];//事件队列
        
        //FULFILLED 状态转换器
        function resolve(value){
            that.value = value;
            that.status = FULFILLED;
            that.queue.forEach(function(item){
            //确定状态,执行事件队列中的 resolve 事件
                item.resolve(that.value)
            })
        }
         //REJECTED 状态转换器
        function reject(reason){
            that.reason = reason;
            that.status = REJECTED;
            that.queue.forEach(function(item){
            //确定状态,执行事件队列中的 reject 事件
                item.reject(that.reason)
            })
        }
        //执行 executor 回调函数,如果在执行过程中遇到错误,则捕获错误并且将状态视为 失败状态发送给 then 的 onRejected 回调函数
        try{
           executor(resolve,reject) 
        }catch(e){
            reject(e)
        }
    }
    
    MyPromise.prototype.then = function(onFulfilled, onRejected){
        if(this.status === FULFILLED){//成功状态则调用 then 的 onFulfilled 回调事件
            onFulfilled(this.value)
        }else if(this.status === REJECTED){//失败状态则调用 then 的 onRejected 回调事件
            onRejected(this.reason)
        }else{//正在进行中则将成功 & 失败 回调存到事件队列缓存中,待确定为成功或者失败状态的时候在执行该事件队列
            this.queue.push({
                resolve: onFulfilled,
                reject: onRejected
            })
        }
    }
    return MyPromise;
})()

# 增加链式操作

也就是说,实际上Promise的then方法返回的也是一个新的Promise对象,这样才可以一直then操作下去.而且then还可以接收上一个then方法return的结果.但是,又有一个问题,上一个then的return结果有如下几种可能:

  1. Promise对象
  2. thenable,即带有then方法的对象
  3. 其他值

关于上一个then的返回值的类型,我们可以创建一个handle函数来接收并且判断,并且确定上一个then方法返回的Promise的状态

大致情况我们基本都清除了,接下来我们只需要改变上面代码中MyPromise.prototype.then方法就行

MyPromise.prototype.then = function(onFulfilled, onRejected){
    var that = this;
    return new MyPromise(function(resolve,reject){
        if(that.status === FULFILLED){//成功状态则调用 then 的 onFulfilled 回调事件
            resHandle(that.value)
        }else if(that.status === REJECTED){//失败状态则调用 then 的 onRejected 回调事件
            errHandle(that.reason)
        }else{//正在进行中则将成功 & 失败 回调存到事件队列缓存中,待确定为成功或者失败状态的时候在执行该事件队列
            that.queue.push({
                resolve: onFulfilled,
                reject: onRejected
            })
        }
        
        function resHandle(value){
            //接收上一个 then 的返回值,如果没有返回值,
            var result = onFulfilled instanceof Function && onFulfilled(value) || value;
            if(result && result.then instanceof Function){
            //当返回值是MyPromise 对象或者是一个 thenable 对象时
                result.then(function(data){
                    resolve(data)
                },function(err){
                    reject(err)
                })
            }else{
                resolve(result)
            }
        }
        
        function errHandle(reason){
            //如果有写 onRejected 回调,则获取回调的返回值,若没有返回值则取 onRejected 的参数
            var err = onRejected instanceof Function && onRejected(reason) || reason;
            if(err && err.then instanceof Function){
             //当返回值是MyPromise 对象或者是一个 thenable 对象时
                err.then(function(data){
                    resolve(data)
                },function(err){
                    reject(err)
                })
            }else{
                reject(err)
            }
        }
    })
}

# Promise.resolve

在《别问，问就得承诺（Promise）》文章中我们明确提到过，关于Promise.resolve的传参方式有4种:

  1. 参数为一个 Promise 实例时,原封不动的将该 Promise 返回出去
  2. 参数为一个 thenable 对象,将其转化为一个新的 Promise 实例
  3. 参数为一个基本类型,返回一个状态为 fulfilled 的新的 Promise,并且将参数 resolve 回调函数中
  4. 无参数,返回一个状态为 fulfilled 的新的 Promise

我们可以直接在 MyPromise 身上改进即可

MyPromise.resolve = function(value){
    //当参数为 Promise 实例时
    if(value instanceof MyPromise){
        return value
    }else if(value && value.then instanceof Function){
        //当参数为 thenable 时
        return new MyPromise(function(resolve,reject){
            value.then(resolve,reject)
        })
    }else{
        //当参数为基本类型 或者 无参数时
         return new MyPromise(function(resolve){
            resolve(value)
        })
    }
}

Promise.reject

Promise.reject()方法的参数，会原封不动地作为reject的理由，变成后续方法的参数。相对于 Promise.resolve来说更为简单

MyPromise.reject = function(err){
    return new MyPromise(function(resolve,reject){
        reject(err)
    })
}

定义Promise.all

Promise.all 方法接收一个参数,参数必须为一个数组,且数组内部成员都必须为一个 Promise 实例对象,当每个 Promise 成员的状态为 fulfilled 时,Promise.all会返回一个 fulfilled 状态的 Promise 对象,假如参数成员中的Promise的状态不全为 fulfilled 的话,则返回一个 rejected 状态的 Promise 对象

MyPromise.all = function(promiseArr){
    if(!Array.isArray(promiseArr)){
        throw Error("Promise.all 参数必须为一个数组")
    }else if(promiseArr.filter(function(item){ 
        return !(item instanceof MyPromise) 
    }).length > 0){
        throw Error("Promise.all 参数数组成员类型必须为 Promise 的实例对象")
    }
    
    var resArr = [],isBreak = false;
    return new MyPromise(function(resolve,reject){
        for(var i = 0,iL = promiseArr.length - 1; i <= iL;i++){
            MyPromise.resolve(promiseArr[i]).then(function(data){
                resArr.push(data)
            },function(err){
                reject(err)
                isBreak = true
            })
            
            if(isBreak){
                break;
            }else{
                resolve(resArr)
            }
        }
    })
}

Promise.race

顾名思义,race有"竞速"的意思,意思就是我就取一个结果,结果是什么不重要,谁最快给我结果我就返回那个结果. 它和 Promise.all 一样,参数为也是接受一个数组且数组成员得全为 Promise 实例

MyPromise.race = function(promiseArr){
	 if(!Array.isArray(promiseArr)){
        throw Error("Promise.all 参数必须为一个数组")
    }else if(promiseArr.filter(function(item){ 
        return !(item instanceof MyPromise) 
    }).length > 0){
        throw Error("Promise.all 参数数组成员类型必须为 Promise 的实例对象")
    }
    
    var done = false;
    return new MyPromise(function(resolve,reject){
    	 for(var i = 0,iL = promiseArr.length - 1; i <= iL;i++){
    	 	promiseArr[i].then(function(data){
    	 			if(!done){
    	 				done = true;
                   		resolve(data)
    	 			}
                },function(err){
                    if(!done){
    	 				done = true;
                   		reject(err)
    	 			}
                }) 
    	 }
    })
}

Promise.prototype.catch

添加一个拒绝(rejection) 回调到当前 promise, 返回一个新的promise。当这个回调函数被调用，新 promise 将以它的返回值来resolve，否则如果当前promise 进入fulfilled状态，则以当前promise的完成结果作为新promise的完成结果.

MyPromise.prototype.catch = function(errFn){
				return this.then(null,errFn)
			}

Promise.prototype.finally

添加一个事件处理回调于当前promise对象，并且在原promise对象解析完毕后，返回一个新的promise对象。回调会在当前promise运行完毕后被调用，无论当前promise的状态是完成(fulfilled)还是失败(rejected)

MyPromise.prototype.finally = function(finallyFn){
				return this.then(finallyFn,finallyFn)
			}
